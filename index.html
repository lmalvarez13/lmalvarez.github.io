Portfolio

Next is a library of games and tools I've been working on.

Medieval RPG

Diablo-like RPG game set in Medieval ambient with its own progression system, weapons arsenal and fantastic atsmosphere.

Enteneo

Trivia game for primary school students about some of Argentina's most dedicated and important scientists throught history. This game is a mix between a fantasy storytelling and biography of these individuals, to help raise interest on their story.
Windows download here!

Shader Project

Final project made for a subject about Computer Graphics & GPUs at the University. Implemented phong's algorithm from scratch as well as some interesting shaders such as cartoony shader and dot/pencil shaders.

Assembly Game Project

Assembly game made using Bochs to run on a IBM-PC with a simple architecture based on Intel AI-32 instructions. Implemented from a basic kernel, scheduler funcionality and syscalls to table pagination and memory managment.

Sonic Dawn

Made this game in 5 days for a Game Jam and as a fresh idea to try and apply new knowledge after having finished CS50G Hardvard Course.
